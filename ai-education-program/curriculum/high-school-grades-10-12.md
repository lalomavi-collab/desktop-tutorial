### יחידה 1: יסודות מדעי המחשב ל-AI (20 שעות)

#### 1.1 Python — שפת ה-AI (10 שעות)
**מטרות למידה:**
- שליטה ביסודות Python הנדרשים לעבודה עם AI
- יכולת כתיבת סקריפטים לעיבוד נתונים

**תכנים:**
- משתנים, תנאים, לולאות, פונקציות
- רשימות, מילונים, מחרוזות
- קריאה וכתיבה של קבצים
- ספריות בסיסיות: NumPy, Pandas
- עבודה ב-Google Colab / Jupyter Notebooks

**פעילויות:**
- סדרת תרגילים מודרכים
- "אתגר Python שבועי"
- תרגיל מסכם: ניתוח Dataset אמיתי

#### 1.2 מתמטיקה ל-AI (10 שעות)
**מטרות למידה:**
- הבנת המתמטיקה הבסיסית מאחורי AI
- חיבור בין מושגים מתמטיים ליישומים ב-AI

**תכנים:**
- אלגברה לינארית בסיסית — וקטורים ומטריצות
- הסתברות בסיסית — הסתברות מותנית, בייס
- סטטיסטיקה — ממוצע, חציון, סטיית תקן
- פונקציות ונגזרות — הבנה אינטואיטיבית
- גרפים ויזואליזציה עם Matplotlib

**פעילויות:**
- "מתמטיקה של Netflix" — הסתברות בהמלצות
- חישוב סטטיסטיקות על Dataset אמיתי
- ויזואליזציה של נתונים ופרשנות

### יחידה 2: למידת מכונה — עיון מעמיק (30 שעות)

#### 2.1 למידה מפוקחת — Supervised Learning (12 שעות)
**מטרות למידה:**
- הבנת אלגוריתמים של למידה מפוקחת
- מימוש אלגוריתמים ב-Python

**תכנים:**
- רגרסיה לינארית (Linear Regression)
- רגרסיה לוגיסטית (Logistic Regression)
- עצי החלטה (Decision Trees)
- K-Nearest Neighbors (KNN)
- שימוש ב-Scikit-learn

**פעילויות:**
- חיזוי מחירי דירות — רגרסיה
- סיווג מיילים לספאם — סיווג
- השוואת ביצועי אלגוריתמים שונים

#### 2.2 למידה בלתי מפוקחת — Unsupervised Learning (8 שעות)
**מטרות למידה:**
- הבנת אשכולות (Clustering) והפחתת ממדים

**תכנים:**
- K-Means Clustering
- אשכול היררכי
- PCA — הפחתת ממדים (רעיון בסיסי)
- יישומים: פילוח לקוחות, זיהוי אנומליות

**פעילויות:**
- פילוח לקוחות של חנות מקוונת
- זיהוי דפוסים בנתוני מזג אוויר
- ויזואליזציה של אשכולות

#### 2.3 הערכת מודלים (10 שעות)
**מטרות למידה:**
- הבנת מדדי הערכה שונים
- ביצוע אופטימיזציה של מודלים

**תכנים:**
- Accuracy, Precision, Recall, F1-Score
- Confusion Matrix
- Cross-Validation
- Overfitting ו-Underfitting — אבחון ופתרון
- Hyperparameter Tuning

**פעילויות:**
- בניית Confusion Matrix ופרשנות
- השוואת מודלים: "איזה מודל הכי טוב?"
- אופטימיזציה של מודל קיים

### יחידה 1: למידה עמוקה — Deep Learning (25 שעות)

#### 1.1 רשתות קונבולוציה — CNN (10 שעות)
**מטרות למידה:**
- הבנת ארכיטקטורת CNN
- בניית CNN לסיווג תמונות

**תכנים:**
- Convolution — הקונבולוציה
- Pooling — דגימה
- ארכיטקטורות מפורסמות: LeNet, AlexNet, VGG (סקירה)
- Transfer Learning — למידת העברה
- Data Augmentation

**פעילויות:**
- בניית CNN מאפס
- שימוש ב-Transfer Learning עם מודל מאומן
- פרויקט: מערכת זיהוי (פנים/צמחים/מזון)

#### 1.2 רשתות רקורסיביות — RNN ו-LSTM (8 שעות)
**מטרות למידה:**
- הבנת עיבוד נתונים סדרתיים
- בניית מודל לניתוח טקסט

**תכנים:**
- מהי רשת רקורסיבית?
- בעיית ה-Vanishing Gradient
- LSTM — Long Short-Term Memory
- יישומים: ניתוח סנטימנט, חיזוי טקסט

**פעילויות:**
- בניית מודל ניתוח סנטימנט
- מחולל טקסט עברי פשוט
- חיזוי מחירי מניות (הדגמה)

#### 1.3 GAN — רשתות יריבות (7 שעות)
**מטרות למידה:**
- הבנת עקרון ה-GAN
- חוויה מעשית עם יצירת תוכן

**תכנים:**
- Generator vs. Discriminator
- תהליך האימון
- יישומים: יצירת תמונות, העברת סגנון, super-resolution
- אתיקה של GANs — Deepfakes

**פעילויות:**
- ניסוי עם StyleGAN
- העברת סגנון אמנותי (Neural Style Transfer)
- דיון: "היכן הגבול?"
