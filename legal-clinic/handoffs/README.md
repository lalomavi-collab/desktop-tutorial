# Semester Handoffs

Per-semester handoff memos generated by `/legal-clinic:semester-handoff`.

## Structure

```
handoffs/
└── [YYYY-term]/           # e.g., 2025-spring / 2025-fall
    ├── _summary.md        # Semester-level summary for professor
    └── [case-id].md       # Per-case handoff memo
```

## Usage

At end of semester, run:

```
/legal-clinic:semester-handoff
```

The skill generates all per-case memos and the semester summary. Professor reviews before passing to incoming students.

Incoming students: run `/legal-clinic:ramp` before accessing handoff documents.

## Confidentiality

Handoff documents are `PRIVILEGED AND CONFIDENTIAL — ATTORNEY WORK PRODUCT`.
Do not share outside the clinic. File names use case IDs only — no client-identifying information.
