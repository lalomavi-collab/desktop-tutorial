# דפוסי תיקים — משרד ללום ושות'

מסמך זה מרכז את הדפוסים החוזרים בתיקי המשרד, לצורך קישור תיקים חדשים למקרים קודמים.

---

## דפוס א' — סכסוך בעלי דירות / שיתוף במקרקעין

**מאפיינים:**
- בניין משותף (מגורים / מעורב)
- מחלוקת על ניצול זכויות בנייה בלתי-מנוצלות
- צדדים: בעלי דירות, חברה מנהלת, קבלן

**חוקים:**
- חוק המקרקעין, תשכ"ט-1969, פרק ד' (בית משותף)
- תקנות המקרקעין (ניהול ורישום)
- חוק התכנון והבניה, תשכ"ה-1965

**כלי המשרד:**
- גישור מכוון-הכרעה (כאשר הסכם שיתוף לא קיים או לא שלם)
- בוררות (כאשר קיים סעיף בוררות בתקנון)
- ליטיגציה (כאשר אין מנגנון הסכמי)

**ספרות:**
- *מדריך לעסקאות תמ"א 38* — אברהם ללום
- *מדריך לעסקאות פינוי-בינוי* — אברהם ללום

---

## דפוס ב' — סכסוך בעלי מניות בחברה קניינית

**מאפיינים:**
- החזקת נכס דרך חברה (SPV / חברת ניהול)
- מחלוקת על חלוקת זכויות, רווחים, או החלטות עסקיות
- פוטנציאל ל-deadlock בין בעלי מניות

**חוקים:**
- חוק החברות, תשנ"ט-1999 (סעיפים 191, 232 — עילות סיום)
- הסכם בעלי מניות (Shareholders Agreement)
- תקנון החברה

**כלי המשרד:**
- גישור מכוון-הכרעה — מומלץ בשלב ראשון
- בוררות — אם קיים סעיף בהסכם
- הליך לפירוק / מינוי מפרק זמני — כמוצא אחרון

---

## דפוס ג' — זכויות בנייה בלתי-מנוצלות (תמ"א / בנייה נוספת)

**מאפיינים:**
- נכס עם פוטנציאל תמ"א 38 / תכנית מפורטת
- חלוקת זכויות לא ברורה בין בעלי הנכס
- קבלן / יזם המבקש לרכוש זכויות

**חוקים:**
- תמ"א 38 (ותיקוניה)
- חוק התכנון והבניה
- תב"ע מקומית (תכנית מתאר גבעתיים)

**כלי המשרד:**
- ניסוח הסכם קומבינציה / תמורות
- ייצוג מול הוועדה המקומית לתכנון ובנייה
- הכנת חוות דעת שמאית לצורכי מו"מ
