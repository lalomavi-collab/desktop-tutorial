## חוק ברזל

> **מותר ליצור קבצים חדשים. אסור לשנות שמות או למחוק קבצים קיימים.**

---

## מבנה תיקיות

```
cases/
  INDEX.md                              ← אינדקס כלל התיקים (תמיד לעדכן)
  patterns/
    README.md                           ← דפוסים חוזרים במשרד
  YYYY_NNN_תיאור_קצר/
    case_file.md                        ← תיק הלקוח הראשי
    documents/                          ← מסמכים שהתקבלו
    correspondence/                     ← תכתובות
```

**שם תיקייה:** `YYYY_NNN_מיקום_נושא` — שנה, מספר סידורי, מיקום, נושא קצר.

## שלב 1 — פתיחת תיק

### 1א. מידע שיש לאסוף מהלקוח בכניסה הראשונה

| שדה | תוכן |
|-----|------|
| שם הלקוח | — |
| סוג ישות | יחיד / חברה / שותפות |
| מיקום הנכס / הסכסוך | — |
| סוג הנכס | מגורים / מסחר / מעורב / קרקע |
| תיאור המקרה בקצרה | — |
| ניסיונות קודמים לפתרון | כן / לא — פירוט |
| לחץ זמן / דדליין ידוע | — |
| האם קיימים מסמכים מיידיים | כן / לא |

### 1ב. יצירת קובץ התיק

צור קובץ `cases/YYYY_NNN_תיאור/case_file.md` עם המבנה:

```markdown
# תיק YYYY-NNN — [נושא]

## נתוני תיק
| שדה | פרטים |
...

## תיאור המקרה
...

## נקודות מפתח לבירור
- [ ] ...

## קישור לדפוסים קיימים
...

## מסגרת משפטית ראשונית
...

## מתווה פעולה מוצע
### שלב א' — איסוף מידע
### שלב ב' — הערכת מצב
### שלב ג' — פעולה

## יומן תכתובות
...

## מסמכים שהתקבלו
...
```

### 1ג. עדכון אינדקס

עדכן את `cases/INDEX.md` — הוסף שורה לתיק החדש בטבלת התיקים הפעילים.

## שלב 2 — קישור לדפוסים קיימים

בדוק את `cases/patterns/README.md` וזהה אילו דפוסים רלוונטיים:

### דפוסי הליבה במשרד

| דפוס | מאפיינים | כלי ראשון |
|------|----------|-----------|
| **א' — שיתוף במקרקעין** | בניין משותף, מחלוקת על זכויות | תקנון + טאבו |
| **ב' — סכסוך בעלי מניות** | SPV / חברת ניהול, deadlock | גישור מכוון-הכרעה |
| **ג' — זכויות בנייה** | תמ"א 38, תב"ע, בנייה על גג | שמאי + ועדה מקומית |
| **ד' — עסקת מכר נדל"ן** | רכישה / מכירה / פינוי-בינוי | נסח, בדיקת שעבודים |
| **ה' — ליטיגציה נדל"ן** | תביעה / בקשת צו / כינוס | הערכת סיכוי + עלות |

ציין בתיק: `רלוונטיות גבוהה / בינונית / נמוכה` לכל דפוס, עם נימוק קצר.

## שלב 3 — מסגרת משפטית ראשונית

לפי סוג הסכסוך, כלול:

**נדל"ן / שיתוף:**
- חוק המקרקעין, תשכ"ט-1969 (במיוחד פרק ד' — בית משותף)
- תקנות המקרקעין (ניהול ורישום)

**חברות / בעלי מניות:**
- חוק החברות, תשנ"ט-1999 (סעיפים 191, 232)
- הסכם בעלי מניות / תקנון

**תכנון ובנייה:**
- חוק התכנון והבניה, תשכ"ה-1965
- תמ"א 38 (אם רלוונטי)
- תב"ע מקומית של הרשות הרלוונטית

**מיסוי:**
- חוק מיסוי מקרקעין, תשכ"ג-1963
- חוק מע"מ (עסקאות מסחריות)

## שלב 4 — המלצה על מסלול פתרון

בהתאם לנתונים, המלץ על אחד מהמסלולים:

| מסלול | מתאים כאשר | כלי המשרד |
|-------|------------|-----------|
| **גישור מכוון-הכרעה** | שני הצדדים רוצים פתרון, ישנו הסכם בסיסי | מתודולוגיה ייחודית של המשרד |
| **בוררות** | קיים סעיף בוררות בהסכם | מינוי בתי משפט — אברהם ללום |
| **מו"מ מובנה** | פתרון אפשרי ללא הליך פורמלי | ניסוח הסכם / תיקון |
| **ליטיגציה** | אין מנגנון הסכמי, סכסוך מהותי | הכנת כתב תביעה / כתב הגנה |
