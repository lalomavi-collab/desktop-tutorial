# ALGO · Legal Billing

מערכת חיוב משפטית אוטומטית עם NLP בצד הלקוח (עברית + אנגלית), בסיס נתוני לקוחות, יומן שעות, וסימולטור חיוב חודשי. הנתונים נשמרים ב-`localStorage` של הדפדפן.

## פיצ'רים עיקריים

- קליטה חכמה (NLP) בעברית ואנגלית - חילוץ לקוח, שעות ותיאור מטקסט חופשי.
- בסיס נתוני לקוחות עם תעריף, אימייל ומתג Auto-Bill.
- יומן שעות עם חישוב אוטומטי של סכום לפי תעריף הלקוח.
- סימולטור חיוב חודשי - מקבץ לפי לקוח ומפיק חשבונית.
- ייצוא חשבונית ל-PDF (תומך עברית RTL מלא דרך html2canvas + jsPDF).
- גיבוי ושחזור JSON של כל הנתונים, ואיפוס דו-שלבי.
- התמדה ב-localStorage.

## הרצה מקומית

דרישה מוקדמת: Node.js גרסה 18 ומעלה (`node -v` לבדיקה).

```bash
npm install
npm run dev
```

הדפדפן ייפתח אוטומטית בכתובת http://localhost:5173

## בנייה לפרודקשן

```bash
npm run build      # יוצר תיקיית dist/
npm run preview    # תצוגה מקדימה של הבנייה
```

לפריסה ב-Vercel: מצביעים על תיקיית הפרויקט, פקודת build היא `npm run build`, ותיקיית הפלט היא `dist`.

## מבנה

- `src/ALGO.jsx` - הקומפוננטה הראשית (כל הלוגיקה: NLP, טבלאות, חיוב, התמדה).
- `src/main.jsx` - נקודת הכניסה של React.
- `index.html` - מעטפת HTML.
- `CLAUDE.md` - הקשר פרויקט עבור Claude Code.

## עבודה עם Claude Code

מהטרמינל, בתוך תיקיית הפרויקט:

```bash
claude
```

Claude Code יקרא אוטומטית את `CLAUDE.md` ויקבל את כל ההקשר על הארכיטקטורה והמוסכמות.

## התמדה (localStorage)

- מפתחות: `algo.clients.v1`, `algo.logs.v1`.
- כל גישה עטופה ב-try/catch - אם האחסון חסום, האפליקציה ממשיכה מהזיכרון.
- כפתור "איפוס נתונים" (דו-שלבי) מוחק את ה-localStorage ומחזיר לברירת המחדל.
