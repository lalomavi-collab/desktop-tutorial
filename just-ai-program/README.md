## חזון אסטרטגי

עולם המשפט חווה מעבר פרדיגמטי — מאימוץ פסיבי של כלי דיגיטל לשילוב **אורגני ועמוק** עם מדעי הנתונים. הבינה המלאכותית אינה רק טכנולוגיה משבשת; היא כוח המעצב מחדש את יסודות הדין, הממשל והאסדרה.

תכנית **Just-AI** מכשירה את **ה"משפטן-טכנולוג"** — מנהיג עתידי שאינו מתכנת, אך משלב:
- ניתוח משפטי קלאסי
- הבנה טכנולוגית עמוקה
- כימות סיכונים אלגוריתמיים
- אתיקה גלובלית ומדיניות

### שנה א׳ — אוריינות טכנולוגית והתנסות
**מטרת-על:** מעבר המשפטן מצופה פסיבי למשתמש אקטיבי
- בניית "שפה משותפת" עם מערכות מבוססות דאטה
- הבנת מכניקת מודלי שפה (LLMs)
- מיומנויות Prompt Engineering לניסוח הנחיות מדויקות
- שימוש בפלטפורמות No-Code ויישומן בעשייה משפטית

### שנה ב׳ — אסטרטגיה, ניהול סיכונים והנהגה
**מטרת-על:** מעבר מהתנסות טכנית להנהגה
- ליטיגציה דיגיטלית והתמודדות עם ראיות מ-AI
- ניהול משברים אלגוריתמיים
- התאמה למדיניות ESG עולמית
- ניווט בסביבה רגולטורית בינלאומית (EU AI Act ומעבר)

## מסגרת הוראה

### שיטות הוראה
| שיטה | אחוז |
|------|------|
| סדנאות מעשיות (Workshops) | 40% |
| Case Studies ומקרי מבחן | 25% |
| הרצאות מומחים (אקדמיה + תעשייה) | 15% |
| עבודות קבוצתיות ופרויקטים | 15% |
| ביקורים ב"חדר בקרה" של חברות הייטק | 5% |

### מרצים
- **מרצי ליבה**: מומחי משפט ו-AI מהאקדמיה
- **מרצים אורחים**: CTOs, Legal Tech VC, רגולטורים (ILITA, ועדת EU)
- **מנטורים**: שותפים בפירמות שאימצו AI, CLOs בחברות טכנולוגיה

## ארבע המסקנות האסטרטגיות

```
1. מאינטואיציה לכימות
   RECIR → משפטן כיועץ אסטרטגי מבוסס-נתונים הניתן לביקורת

2. אוריינות כהכרח קיומי
   עורכי דין שלא יאמצו AI → שוליים תחרותיים

3. שומר הסף האתי
   בעידן מערכות אוטונומיות → תפקיד משפטי קריטי מאי פעם

4. עליונות אסטרטגית
   מומחיות משפטית + Deep Tech → ישראל מובילה LawTech 2030
```
